# Nvim Config & Tmux

#### install vim plug
```bash
curl -fLo ~/.local/share/nvim/site/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

#### install python dependencies
```bash
pip install neovim
pip install jedi
pip install yapf
pip install pylint
```
